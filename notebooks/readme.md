# Movie Recommender System

A content-based movie recommendation system built using Python.
This project uses movie metadata to recommend similar movies based
on textual features such as overview, genres, and cast.

## Tech Stack

- Python
- Pandas & NumPy
- Scikit-learn
- Streamlit

## Features

- Content-based filtering
- Cosine similarity
- Interactive web interface with Streamlit

## Dataset

This project uses the TMDB dataset from Kaggle.
Dataset files are not included in the repository.
